# Task: Web Frontend

## Description

Implement a minimal web frontend with input form and result display. Use plain HTML, CSS, and vanilla JavaScript—no frameworks. Embed static files in Go binary.

## Acceptance Criteria

- [x] Input form for submitting ideas
- [x] Loading state during pipeline execution
- [x] Display decision.json in formatted view
- [x] Display todo.md as rendered markdown
- [x] Bilingual UI (Chinese/English toggle)
- [x] Responsive design for mobile
- [x] Embedded in Go binary via embed package

## Technical Details

### File Structure
```
web/
  static/
    index.html
    styles.css
    app.js
    i18n.js      # Internationalization
```

### UI Components
1. **Header** - Title, language toggle
2. **Input Form** - Textarea, submit button
3. **Loading State** - Spinner, "Processing..." message
4. **Results Panel**
   - Decision card (ruling, rationale, rejected)
   - Todo list (rendered markdown with checkboxes)
5. **Error Display** - Error message card

### API Integration
```javascript
async function submitIdea(input) {
    const response = await fetch('/api/verdict', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ input })
    });
    return response.json();
}
```

### Embedding in Go
```go
//go:embed static/*
var staticFiles embed.FS

func main() {
    r.Handle("/static/*", http.FileServer(http.FS(staticFiles)))
    r.Get("/", func(w http.ResponseWriter, r *http.Request) {
        http.ServeFile(w, r, "web/static/index.html")
    })
}
```
